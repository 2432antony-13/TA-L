# 🔮 手势塔罗项目（MysticGesticTarot）

> 一个基于手势控制的现代塔罗牌交互应用

---

## 📋 项目概述

**核心理念**：通过手势识别技术，让用户以更自然、沉浸式的方式体验塔罗占卜。

### 技术栈
- **前端**: React + TypeScript + Vite
- **样式**: TailwindCSS v4
- **视觉**: CSS 滤镜增强 + AI 图像放大
- **塔罗牌**: Rider-Waite 1909 经典牌组

---

## 📣 营销定位

**产品故事**：「塔罗叙事」—— 用手势唤醒古老的智慧

**核心卖点**：
1. **沉浸式体验** - 手势控制，仿佛亲手触摸命运
2. **经典传承** - 采用1909年韦特塔罗经典图案
3. **现代美学** - 高清增强，视觉震撼
4. **隐私安全** - 本地运算，数据不上传
